How to properly utilize this course

  1. Only access modules through this Dashboard (Home page)
  2. Do not use the
    Previous
    and
    Next
    buttons to navigate the course
  3. When completing a module, do it in one sitting as progress is not recorded for semi-complete modules
  4. In order for your progress to be valid you must score a perfect grade in every quiz (you have unlimited retries)
