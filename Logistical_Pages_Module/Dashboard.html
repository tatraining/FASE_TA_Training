Please note that most of the content in this course requires that you open Quercus on a desktop or mobile browser
Only the "TA Resources" page is fully supported on the "Canvas Student" and "Canvas Teacher" apps
 
 

How to properly use this course

  1. Only access modules through this Dashboard (Home page)
  2. Do not use the
    Previous
    and
    Next
    buttons to navigate the course
  3. Use the
    Continue
    and
    Back to Dashboard
    buttons to navigate the course
  4. When completing a module, do it in one sitting as progress is not saved for semi-complete modules
  5. In order for your progress to be valid, you must achieve 100% in every quiz (you have unlimited retries)
  6. Please keep track of the modules you have completed, as this will not be recorded.
